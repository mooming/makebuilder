PROJECT OVERVIEW:
MakeBuilder is a C++23 CMake-based meta-build system that automatically generates
CMakeLists.txt files based on project directory structure. It supports various
build types (Executable, StaticLibrary, SharedLibrary, HeaderOnly, etc.) and
provides cross-platform support for Windows and POSIX systems.
